About P2P WEB

P2PWeb is a framework that attempts to make building peer-to-peer web apps a breeze. This website serves as a demo for that framework which allows users to store, retreive, and discover content in a completely peer-to-peer way. All the posts you see below are not stored on any server and instead served straight from the users visiting the website. P2PWeb is in an early alpha state so bugs are likely and load times are slow, so please allow a few seconds for the content to load from your peers.
